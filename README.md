# 🧹 SwipeCleaner

> A fast, keyboard-driven macOS app to clean thousands of photos and videos effortlessly.

---

## ✨ Why I Built This

Managing large photo/video collections is painful. Screenshots, duplicates, blurry shots… they pile up fast.

I built SwipeCleaner to solve this for myself:

- Quickly review media
- Decide instantly: keep or delete
- Stay focused with a keyboard-only workflow

---

## 🚀 Features

- ⚡ Lightning-fast navigation
- ⌨️ Keyboard-first experience
- 🗂️ Source → Target workflow
- 🗑️ Safe deletion (moves to Trash)
- 📦 Bulk delete skipped items

---

## 🎮 Controls

| Key | Action |
|-----|--------|
| `D` | Delete (move to Trash) |
| `K` | Keep (move to Target folder) |
| `←` / `→` | Navigate between items |
| `X` | Delete all skipped items |

---

## 🧠 How It Works

1. **Select a Source Folder** — where your messy files are
2. **Select a Target Folder** — where kept files go
3. **Start reviewing:**
   - **Keep** → moved to target folder
   - **Delete** → moved to Trash
4. Optionally press `X` to bulk-delete everything you skipped

---

## 🖼️ Supported Media

- Images (JPG, PNG, etc.)
- Videos (MP4, MOV, etc.)

---

## 🛠️ Tech Stack

- Swift
- SwiftUI / AppKit
